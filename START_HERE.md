## ⚙️ **АВТОМАТИЧЕСКИЕ СКРИПТЫ:**

### **Windows (PowerShell):**
```powershell
$env:DATABASE_URL="ваш-url-из-vercel"
.\setup-database.ps1
```

### **Mac/Linux (Bash):**
```bash
export DATABASE_URL="ваш-url-из-vercel"
chmod +x setup-database.sh
./setup-database.sh
```
